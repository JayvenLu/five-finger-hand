# Five-Finger Dexterous Hand CAD

This directory contains the editable STEP design sources.

## Files

| Source file | Description |
| --- | --- |
| `mold.STEP` | Fingertip mold assembly |
| `thumb.STEP` | Thumb component |

Use the STEP files for manufacturing, inspection, and modification.
